# HealthFlow - Symptom-Based Pharmaceutical Marketplace

HealthFlow is a modern B2C e-commerce platform designed for the healthcare market. It helps users discover and purchase pharmaceutical products based on specific symptoms.

## 🚀 Key Features
- **Symptom Mapping:** Interactive checkbox-based UI to identify health concerns.
- **Product Recommendations:** Automatic suggestion of relevant medicines.
- **E-Commerce Suite:** Fully functional cart and checkout system.
- **Admin Dashboard:** Inventory management and order tracking.
- **Payment Method:** Integrated Cash-on-Delivery (COD) workflow.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Python (Flask)
- **Database:** MySQL
